    • Cuerpo principal del contenido

    La etiqueta main es en HTML5 una etiqueta semantica. Esto quiere decir que si tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el domcumento. El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de pagina, los menuies de navegacion y los pue de pagina). Es usado por buscadores para indexar. Conviene usar aqui textos que atuden a encontrar la pagina luego de procesos de indexacion de los buscadores.

    Seccion etiquetas semanticas estructurales mas importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADRESS
    • Las etiquetas semanticas estructurales estan relacionas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

    Reglas para etiquetas semanticas:

  • Los distintos tipos de contenidos se puedenn asociar a la etiqueta SECTION
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. o sea que una SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER Y FOOTER pueden ser anidadeas dentro de otras HEADER O FOOTER.Osea un FOOTER podria anidar a otro HEADER Y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento.
  • Reservar etiqueta h1 para usa dentro de etiquetas top level como main,header o footer.
